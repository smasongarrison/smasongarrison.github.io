rareGEphenoR Documentation

Example phenotype data

Description

Example phenotype data for rareGE.

Usage

data(rareGEpheno)

Format

A data frame with 500 observations on the following 5 variables.

y1

a quantitative trait.

y2

a dichotomous trait.

age

a quantitative covariate.

sex

a dichotomous covariate.

bmi

a quantitative covariate.
