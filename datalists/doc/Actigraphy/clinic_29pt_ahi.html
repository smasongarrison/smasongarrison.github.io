clinic_29pt_ahiR Documentation

Data Set Containing AHI(Apnea Hypopnea Index) Information from 29 Subjects

Description

A data set containing the AHI(Apnea Hypopnea Index) values(continuous) for 29 subjects.

Usage

data(clinic_29pt_ahi)

Format

A data frame consisting of two columns. The first column contains the numeric IDs and the second column contains the AHI information of the 29 subjects.
