act_29ptR Documentation

Data Set Containing a One Day Average of Actigraph Data for 29 Subjects

Description

A data set containing a one day(1440 minutes) of actigraph data for 29 subjects.

Usage

data(act_29pt)

Format

A data frame that consists of 30 columns; the first column refers to the time indices of one day by minute from midnight to midnight. The following 29 columns contain the actigraph data of 29 subjects with column names being the subject IDs.
