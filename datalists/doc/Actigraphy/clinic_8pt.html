clinic_8ptR Documentation

Data Set Containing AHI(Apnea Hypopnea Index) Information from 8 Subjects

Description

A data set containing the AHI(Apnea Hypopnea Index) values(categorical) for 8 subjects.

Usage

data(clinic_8pt)

Format

A data frame that consists of two columns. The first contains the numeric IDs and the second column contains the AHI information for 8 subjects.
