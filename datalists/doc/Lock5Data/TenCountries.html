TenCountriesR Documentation

Ten Countries

Description

A subset of the ALLCountries data for a random sample of ten countries

Format

A dataset with 10 observations on the following 4 variables.

Country Armenia Bahamas Lebanon Macedonia North Korea Romania Serbia Slovenia Tunisia Uzbekistan
Code ARM BHS LBN MKD PRK ROU SRB SVN TUN UZB
Area Area (in 1,000's sq km)
PctRural Percentage of population living in rural areas

Details

A random sample of ten countries selected from the AllCountries dataset.

Source

Data collected from the World Bank website, worldbank.org.
