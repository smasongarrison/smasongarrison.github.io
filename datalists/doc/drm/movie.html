movieR Documentation

Movie reviews from a Variety magazine

Description

This dataset contains reviews of 93 movies by four critics in the Variety magazine from April 1995 through March 1997.

Usage

data(movie)

Format

This data frame contains the following columns:

movie

Identification number of a movie.

y

review response, 1: Positive; 2: Mixture of positive and negative; 3: Negative.

critic

an ordered factor: critics with levels medved < siskel < ebert < lyons

Source

Hartzel, J., Agresti, A., Caffo, B. (2001). Multinomial logit random effects models. Statistical Modelling, 1: 81-102
