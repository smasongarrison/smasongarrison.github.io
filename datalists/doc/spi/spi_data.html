spi_dataR Documentation

SPI of the rainy season in the Rio Grande do Norte

Description

This data set present the Standardized Precipitation Index (SPI) of the rainy season in the Rio Grande do Norte State - Brazil.

Usage

spi_data

Format

A matrix containing 336 observations.

Source

Empresa de Pesquisa Agropecuaria do Rio Grande do Norte - EMPARN

References

EMPARN. Rainfall database of the Rio Grande do Norte. URL http://www.emparn.rn.gov.br
