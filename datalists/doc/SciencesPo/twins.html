twinsR Documentation

Burt's Twin Data

Description

The given data are IQ scores from identical twins; one raised in a foster home, and the other raised by birth parents. This dataset contains the following columns:

  • C Social class, C1=high, C2=medium, C3=low, a factor.

  • IQb biological.

  • IQf foster.

@references Weisberg, S. (2014). Applied Linear Regression, 4th edition. Hoboken NJ: Wiley.

Usage

data(twins)

Format

A data.frame object with 3 variables and 27 observations.

Source

Burt, C. (1966). The genetic estimation of differences in intelligence: A study of monozygotic twins reared together and apart. Br. J. Psych., 57, 147-153.
