simesantsR Documentation

Harvard Forest Simes Tract Ant Community Data

Description

Within the Simes tract of Harvard Forest, we sampled ants by hand collection along 100-m transects at eight sites with the following microhabitats: two sites each of hemlock, white pine, and hardwood overstories; a transition between hardwood swamp and pine; and a rocky-slope with a hardwood overstory.

Usage

data(simesants)

Details

Columns correspond to the ant species, designated by a six letter code combing the first three letters of the genus with the first three letters of the species.
